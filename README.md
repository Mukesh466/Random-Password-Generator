# ADJUATABLE RANDOM PASSWORD GENERATOR

# AIM
  A simple and secure Password Generator built with Python that creates strong random passwords using alphabets, numbers, and symbols based on user-defined percentages.

# FEATURES USED
  Generate secure random passwords
  
User-defined password length

Percentage-based character distribution

Alphabets %

Numbers %

Symbols %

Random password shuffling

Strong password generation logic

# TECHNOLOGIES USED
 Python
 
random module

string module

# THE PROGRAM

Takes password length from the user

Calculates:

Alphabet count

Number count

Symbol count

Generates random characters

Shuffles them for better security

Displays the final password

# Future Improvements

GUI using Tkinter

Password strength checker

Copy to clipboard feature

Save generated passwords

Dark/light mode UI
